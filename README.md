## üìñ About
**Juno** is a dedicated platform designed for college students to discover and apply for open positions within various campus clubs. It helps clubs post their opportunities and manage applications, while providing students with an easy way to explore, apply, and connect with their college community.

## ‚ú® Key Features

### üéì For Students
* **Smart Discovery:** Search for club openings by keywords, categories, or required skills using the dedicated "Opportunities" page.
* **Seamless Application:** Apply directly within the platform. Upload resumes, cover letters, and provide specific details requested by the club.
* **Bookmark Opportunities:** Not ready to apply? Save listings to your "Saved Opportunities" section to review later.
* **Application Tracking:** Monitor your progress from "Submitted" to "Accepted" or "Declined" via the "My Applications" dashboard.

### üèõÔ∏è For Clubs & Organizations
* **Easy Recruitment:** Quickly post openings by navigating to the "Create Opening" section.
* **Custom Listings:** specific details about the position, club culture, and requirements.
* **Management Dashboard:** (Implied) Manage incoming applications and streamline the selection process.

## üöÄ Getting Started

### Prerequisites
* Node.js
* npm / yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/yourusername/juno.git](https://github.com/yourusername/juno.git)
    cd juno
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Set up environment variables**
    Create a `.env` file in the root directory and add the necessary variables:
    ```
    VITE_SUPABASE_URL=
    VITE_SUPABASE_ANON_KEY=
    VITE_CLERK_PUBLISHABLE_KEY=
    ```

4.  **Run the application**
    ```bash
    npm run dev
    ```

## üí° Usage Guide

### How to Post an Opening (Club Admin)
1.  Log in as a Club Representative.
2.  Navigate to the **"Create Opening"** section.
3.  Fill in the position details, requirements, and club info.
4.  Submit the form to make the listing live.

### How to Apply (Students)
1.  Browse openings via the Search Bar or Filters.
2.  Click on a listing to view details.
3.  Follow the instructions (Upload Resume/Cover Letter).
4.  Click **Submit**.
5.  Check the **"My Applications"** section for status updates.
